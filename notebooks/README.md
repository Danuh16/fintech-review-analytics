# notebooks/

Place exploratory Jupyter notebooks here. Each notebook should correspond to a task:

- `01_data_exploration.ipynb` — Task 1: inspect raw and clean datasets
- `02_sentiment_analysis.ipynb` — Task 2: sentiment & theme exploration
- `03_database.ipynb` — Task 3: database schema and queries
- `04_insights.ipynb` — Task 4: visualisations and final insights

Notebooks are for exploration only. Production logic lives in `scripts/` and `src/`.
